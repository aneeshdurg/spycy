sPyCy browser demo

Input openCypher queries and use `ctrl + enter` to execute them!

This project is open source!

Sample Queries
  • Create nodes with ids 1 to 5:
    • UNWIND range(1, 5) AS id CREATE (n:Node {id: id})
  • Return all nodes:
    • MATCH (n:Node) RETURN n, properties(n)
  • Return nodes with id > 3:
    • MATCH (n:Node) WHERE n.id > 3 RETURN n, properties(n)
  • Create a relationship between nodes with even and odd ids:
    • MATCH (n: Node) WHERE n.id % 2 = 0 CREATE (n)-[:EVEN]->(m:Node {id: n.id + 1})
  • Return all relationships:
    • MATCH (n:Node)-[r]->(m) RETURN n.id, r, m.id
